Contrast set learning

Usage:

    rast : Contrast set learning

The difference between things (in the same domain) is

USAGE: ./rast.lua [OPTIONS]

OPTIONS:

  -data FILE    read data from FILE
  -goal STR     one of optimizer, monitor, explode
